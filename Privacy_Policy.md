Privacy Policy for ClaudeShift

Effective Date: May 3, 2026

ClaudeShift is a browser extension designed for Google Chrome that helps users manage and switch between multiple Claude.ai sessions. This policy explains what data is accessed, how it is used, and how it is protected.

---

1. Data Collection

ClaudeShift does not collect, store, or transmit personal data to any external servers.

The extension accesses the following data locally within your browser:

* Cookies associated with https://claude.ai for session switching
* Session data required to save and restore accounts
* User preferences stored using Chrome storage

ClaudeShift does not collect login credentials such as usernames or passwords.

All accessed data remains on your device and is never transmitted externally.

---

2. Use of Data

The extension uses data only to:

* Save active Claude.ai sessions
* Restore sessions when switching accounts
* Enable fast account switching functionality

No data is used for tracking, analytics, profiling, or advertising.

---

3. Data Sharing

ClaudeShift does not:

* Sell user data
* Share data with third parties
* Transfer any data outside the user's browser

All operations are performed locally.

---

4. Permissions Justification

ClaudeShift requests the following permissions:

* cookies: To read and modify Claude.ai session cookies for account switching
* storage: To store session data locally on the device
* tabs: To detect and refresh Claude.ai tabs
* scripting: To interact with Claude.ai pages when switching sessions
* alarms: To support background session handling if required

Permissions are used strictly for core functionality.

---

5. Data Security

All data is stored locally using Chrome's secure storage system.

No external servers, APIs, or remote databases are used.

---

6. Third-Party Services

ClaudeShift does not use third-party services, analytics tools, or external APIs.

---

7. User Control

You have full control over your data:

* You can delete saved sessions at any time
* You can clear extension storage through browser settings
* You can uninstall the extension to remove all stored data

---

8. Changes to This Policy

This policy may be updated if the extension’s functionality changes. Updates will include a revised effective date.
