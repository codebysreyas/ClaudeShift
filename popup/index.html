Why it exists

Built for people who use Claude seriously.

Researchers, developers, and anyone juggling work and personal accounts shouldn't have to log in twice. ClaudeShift removes that friction entirely.

01

One-click switching

Select a saved session from the popup. The tab refreshes and you're in. No login form, no waiting, no friction.

02

Entirely local

Session data is stored using Chrome's built-in storage API. Nothing is sent to any server — not ours, not anyone's.

03

Unlimited profiles

Save as many Claude accounts as you need — work, personal, client projects. Each one is a single click away.

Installation

Running in 30 seconds.

  1. 01

    Download the extension

    Clone or download this repository to your machine. No build step required — it's plain JavaScript.

  2. 02

    Load it into Chrome

    Enable Developer Mode in Chrome extensions, then click Load unpacked and select the project folder.

    chrome://extensions/
  3. 03

    Save your first session

    Open claude.ai, log into your first account, click the ClaudeShift icon, and hit Save Session.

  4. 04

    Switch between accounts

    Repeat for each account. From then on, switching is a single click from the extension popup — any time, on any Claude tab.

No data collection. No tracking. No external servers. ClaudeShift uses Chrome's local storage API. Your session data never leaves your device and is never accessible to anyone but you.
